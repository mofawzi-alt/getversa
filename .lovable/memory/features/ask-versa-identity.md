Ask Versa is PUBLIC SENTIMENT INTELLIGENCE, not a generic AI assistant.

## Core Principles
1. **Poll-backed answers first** — lead with real percentages, demographics, vote counts
2. **AI supports, never dominates** — AI summarizes/interprets data, never overshadows real votes
3. **Socially insightful** — answers should be surprising, comparative, identity-based, tribal
4. **Public sentiment feel** — "checking Egypt's pulse", not "reading AI output"
5. **Trust & authenticity** — always show data provenance (vote count, poll count, demographics)

## When no poll data exists
- Be honest: "No direct poll data on this yet"
- Never fake it with AI-generated opinions
- Suggest specific A-vs-B questions to ask instead
- Offer "Suggest a Poll" CTA

## Tone
- Like discovering what real people actually think
- Social contrasts, tribal differences, identity tension
- Never academic, never survey-report-like
- Frame demographic splits as revelations, not tables

## Labels
- "Egypt voted" (not "Versa says")
- "Social Insight", "Who Disagrees", "Egypt's Pulse", "The Verdict"
- "Checking real votes…" (loading state)
- "No direct poll data yet" (smart_answer fallback)
